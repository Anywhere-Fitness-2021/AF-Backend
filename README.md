# AF-Backend
Node/Express/SQLite/Knex Backend For Unit 4

CLASSES ENDPOINTS:

[GET] All Classes

[GET] Class By Id

[PUT] / Update Class By ClassId

[POST] New Class

[DELETE] Class By ClassId

USERS ENDPOINTS:

[POST] Registration Of New User

[POST] Login Of User
